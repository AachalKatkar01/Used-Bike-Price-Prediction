# Used Bike Price Prediction

## 📌 Project Overview
This project predicts the resale price of used bikes using Linear Regression. The dataset includes features like model year, kilometers driven, and number of previous owners.

## 📊 Exploratory Data Analysis
- Price Distribution
- KMs Driven vs Price
- Owner Type vs Price

## 🤖 Machine Learning Model
- Algorithm: Linear Regression
- Train-Test Split: 80-20
- Evaluation Metrics:
    - Mean Squared Error (MSE) : 15441903660.994251
    - R2 Score :0.06790221186311607

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
